ABOUT SECTION (of the projects):


Expi:

Expi is a bot I created for my friend Newmentor for his discord server, it comes with several commands that connect his players more towards the game. I had a lot of fun creating this discord bot and have been developing on it since 23/10/2017 (October the 23rd), this bot has been my first project using discord.js and node.js. It was not an easy and straight forward road, there were many hilarious bumps and annoyances with it. Many said that I should of studied Javascript more in able to create better code and less mistakes but, to me, creating a project allows more learning and you are more likely to remember the language better!
Extra Links that this project is involved in: RobloxManjaro, Official Roblox Kilimanjaro Blog


ThinzyBOT:

 My bot was created a long LONG time ago (in a galaxy far away, jk), before I even knew how/what language to code it with, the commands are basically a clone of Expi as I haven't had as much time to develop on it properly due to my schedule at the minute. This one shall later be developed on more than Expi due to it being my presonal Discord Bot, it will definitely be a stable project (hopefully haha) as I wish to intergrate some good commands that shall make my discord server more lively.
